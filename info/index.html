How to edit native information?

You also may noticed that there is no edit function at the moment. This is because it simply isn't implemented yet, in any case some kind of review system might be a better idea so the database doent get destroyed like the original! If you still really want to edit some information of a native please visit this repository. In order to edit something you will first need to fork the repo and make the changes you'd like, then simply create a pull request for review. Once reviewed it can be merged into the main repository.


I have an idea for the website! Where do I suggest it?

If you know how to program then simply fork the github repository, make your changes and create a pull request! If you don't know how to program but still want to notify me about your suggestion, please create a new issue on the websites github repository!
