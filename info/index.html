How to edit native information?

You also may noticed that there is no edit function at the moment. This is because we simple didn't implement it yet because we want some kind of review system so no one can destroy anything! If you still really want to edit some information of a native please visit this repository. In order to edit something you first need to fork the repo and make the changes you'd like to do. After that is done you simply create a pull request and we'll review it. Once we review and merged your changes the website updates automatically.


I have an idea for the website! Where do I suggest it?

If you know how to program then simply fork the github repository, make your changes and create a pull request! If you don't know how to program but still want to notify us about your suggestion, please create a new issue on the websites github repository!
